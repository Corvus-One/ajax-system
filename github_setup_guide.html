Part 2 — Upload Your AJAX File

Part 4 — Your Permanent URL
6
Bookmark Your Live URL
This is your permanent address. It never expires. Bookmark it and you're done.
https://corvus-one.github.io/ajax-system/

Open that URL in Chrome. You should see the AJAX system with the yellow API key banner at the top. Paste your Anthropic API key, click Save Key, and every agent is ready to run.

Your API key stays in your browser only. It is never uploaded to GitHub, never visible in your file, and never sent anywhere except directly to the Anthropic API when you run an agent.
Part 5 — Updating the System Later
7
How to Update When Claude Delivers a New Version
Every time we ship an improvement, follow these 3 steps to push it live.
  1. Download the new file from Claude — save it as index.html
  2. Go to your repo — github.com/corvus-one/ajax-system — click on index.html in the file list
  3. Click the pencil icon (Edit) → then click Upload file at the top of the editor → drop your new index.html → Commit changes. Live in 60 seconds.
Your data is safe across updates. Profile, pipeline applications, and API key are all stored in your browser's localStorage — not in the HTML file. Updating the file never wipes your data.
Troubleshooting
?
Common Issues & Fixes
If something doesn't look right, start here.
Page shows 404 error: GitHub Pages isn't live yet — wait 90 seconds and refresh. If it persists, go back to Settings → Pages and confirm Branch is set to "main" and folder is "/ (root)".
Page is blank or shows raw code: The file wasn't saved as index.html. Go to your repo, click the filename, rename it to index.html using the pencil icon.
Agents still say "network error": Clear your browser cache (Ctrl+Shift+Delete) and reload. Then re-enter your API key — it clears with the cache.
"Invalid API key" error: Your key may have expired or was entered incorrectly. Go to console.anthropic.com → API Keys → create a new one and paste it fresh.
Everything working: Green bar says "✓ API KEY CONFIGURED" and the first agent run returns a real response. Bookmark the URL and you're operational.
